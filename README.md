Software and tool Used:

Python Flask Framework

Llama-3: The Model (The Brain): Llama 3 or Mistral. These are the "intelligence" that can understand and generate text.

Langchain: The Agent (The Worker): This is what you build using a framework like LangChain. An agent is a "manager" that uses the brain (running on Groq) to take actions, like searching the web or checking a database.

Groq: The Platform (The Engine): Groq. It provides the specialized chips (LPUs) that make those brains think and speak instantly.
